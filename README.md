# Calculator with ReactJS

> A Todo list using ReactJS. You can add your daily plans and remove them.


## Built With

- HTML, CSS & JavaScript.
- webpack, ReactJS


## live preview
Not yet available.


## Getting Started

### Prerequisites

> Any Web Browser

### if you want to setup the project
- clone the project
- `cd Todo-list-with-React`
- `npm i`
- `npm start`
